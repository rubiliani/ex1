Students Web-service API

EX 1 - ws course, this web-service allows you to get students data from mlab db

Instructions

Get all students

Method: GET
URL Example: /api/students
Response: 
[
    {
        "firstname": "Rubi",
        "lastname": "Liani",
        "id": "1234",
        "grade": 91,
        "year": 2016
    }
    ...
]

Get student grade by ID

Method: GET
Param: id
URL Example: /api/students/grade/1234
Response: 
[
    {
        "grade": 91
    }
    
]

Get all excellence students (students with grade 90 and above)

Method: GET
URL Example: /api/students/excellence
Response: students array as above

Get all excellence students by Year (students with grade 90 and above)

Method: GET
Param: year
URL Example: /api/students/excellence/2016
Response: students array as above
